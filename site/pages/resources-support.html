Best For

Best when rapid diagnosis is needed and failure must be isolated with reproducible evidence.

  • Collect environment/version details first
  • Use targeted diagnostics scripts
  • Escalate with exact reproduction steps

Anti-Patterns To Avoid

These behaviors reduce framework signal and increase rework risk.

  • Escalating without logs
  • Patching symptoms not root cause
  • Skipping version/platform checks

Adoption Plan

  1. Capture current state and error output
  2. Run install/update diagnostics
  3. Check adapter files and gates
  4. Escalate with minimal reproducer
